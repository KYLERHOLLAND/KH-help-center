## What to include

Copy this checklist into your message so we can reproduce the issue faster:

1. **Subject line:** `Bug: [Product name]` (or `Feature request: [Product name]`)
2. **Purchase email** — the address used at checkout
3. **Product name** — e.g. Auto Resize Lower Third, 200+ Text Effects
4. **App and version** — e.g. Premiere Pro 25.2, Lightroom Classic 14.1
5. **Operating system** — e.g. macOS 15, Windows 11
6. **What you expected** vs **what actually happened**
7. **Steps to reproduce** — numbered steps if you can
8. **Screenshots or a short screen recording** — especially any error messages
