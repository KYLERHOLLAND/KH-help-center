## Installation

### Step 1 — Download and extract

1. Download the product `.zip` from your order email or [your account](https://assets.kylerholland.com/u/signin/).
2. Extract the `.zip` file.
3. Locate the `.mogrt` file inside.

### Step 2 — Install in Premiere Pro

1. Open **Adobe Premiere Pro**.
2. Open the **Essential Graphics** panel (**Window** → **Essential Graphics**).
3. Drag the `.mogrt` file into the Essential Graphics panel, or click **Install Motion Graphics Template** and select the file.

The template appears under **Local Templates** and is ready to drag onto your timeline.

## How to use

### Scale Width Up + Down workflow

1. Drag the MOGRT onto the timeline over your video, then copy it across every text beat you need (hold **Option/Alt** and drag to duplicate).
2. Trim each MOGRT clip to match the video segment it covers.
3. Open each MOGRT and change the text (for example: SUBSCRIBE → TO → KYLER → HOLLAND).
4. Set **Intro** and **Outro** settings on each MOGRT:
   - **Easy way:** Use the same intro/outro on every clip — transitions still look clean.
   - **Best results:** First clip uses **No Intro**. Match each clip's outro to the next clip's intro (Scale Width Up when the next word is larger, Scale Width Down when it's smaller). Last clip uses **No Outro**.
