## Installation

1. Download and **extract** the zip to a folder you'll remember (the tutorial stresses an easy-to-find location).
2. Premiere → **Effects** → right-click **Presets** → **Import Presets**.
3. Navigate to the extracted preset file → **Open**.

---

## How to use

1. Select an **audio clip** (or multiple clips) on the timeline.
2. Drag a preset from the **Audio Effects** bin onto the selection.
3. Open **Effect Controls** / **Audio Track Mixer** to adjust levels after the preset loads.
4. **Remove Attributes** on the clip if you want to try a different chain.

Unlike video transition packs, these apply to **audio only** — no adjustment layer required.

---

## Tips and workflow

- **Gain-stage first** — normalize rough levels before heavy processing presets.
- **Solo preview** — solo the track when comparing presets so other tracks don't mask changes.
- **Save duplicates** — copy a clip with a preset you like as a template for the rest of the interview.
