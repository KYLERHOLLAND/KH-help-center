Installation

  1. Download and extract the product .zip.
  2. In Premiere Pro, open the Effects panel.
  3. Right-click Presets → Import Presets → select the Seamless Subtitles .prfpset.
  4. Expand the imported bin to browse animation categories.

How to use

Fast workflow (as shown in the tutorial)

  1. Create or import captions on your timeline (for example auto-generate captions for a social clip in Premiere).
  2. Select all caption blocks you want styled in one pass (click the first, Shift-click the last, or marquee-select on the timeline).
  3. Drag a preset from the Seamless Subtitles bin onto the selection — animations apply to every selected caption layer at once.
  4. Move to another clip, select its captions, and drag a different preset for variety.

No per-line keyframing required for the base look; swap presets until you find the style that fits the edit.

Single-caption tweaks

  1. Select one caption layer.
  2. Apply one preset, preview, then Remove Attributes if you want to try another.
  3. Open Effect Controls or the text panel to adjust font/size after the preset is applied.

Tips and workflow

  • Select before you drag — bulk selection is the fastest way to match TikTok/Reels caption pacing across a clip.
  • Mix presets per scene — different presets on adjacent social clips keep the edit from feeling repetitive.
  • Pair with Brevidy — generate captions with Brevidy, then use Seamless Subtitles presets for motion styling.
  • Try the free sample if you're unsure which animations fit your brand.

Frequently asked questions

Do these work on Essential Graphics titles?

They're optimized for caption/subtitle text layers. MOGRT titles may work in some cases, but the tutorial workflow targets Premiere captions.

Can I animate one word at a time?

The speed demo applies presets to blocks of captions you select. For word-by-word custom motion, adjust timing after applying or edit individual caption spans.
