## Installation

1. Extract the downloaded `.zip`.
2. **Effects** → **Presets** → right-click → **Import Presets** → select the VHS `.prfpset`.
3. Open the imported bin before your edit session so presets are easy to grab.

---

## How to use

1. Add an **adjustment layer** at the cut (or follow clip-based placement in the video).
2. Drag a **VHS Glitch** preset onto the layer.
3. Trim layer length to control how long the glitch sustains.
4. Combine with [10 Glitch Overlays](10-glitch-overlays/index.md) on graphics tracks for layered retro looks.

The tutorial stresses **drag-and-drop** speed — preview several presets on the same cut before committing.

---

## Tips and workflow

- **Music videos & social** — short VHS hits between beats read cleaner than long sustained glitch.
- **Render preview** — heavy glitch stacks may need rendering for smooth scrubbing.
- **Don't overuse** — one or two VHS moments per minute usually carries the aesthetic.
