## Installation

1. Extract the `.zip`.
2. **Effects** → right-click **Presets** → **Import Presets**.
3. Select **KH Text Easy Slide Preset Pack** (`.prfpset`) → **Open**.

---

## How to use

1. Create a **text layer** or **Essential Graphics** title on your timeline.
2. Trim the text clip to the duration you need on screen (short social supers or long lower thirds).
3. Drag a slide preset from the **Text Easy Slide** bin onto the text clip.
4. Preview — the animation scales with the clip length because the pack is **duration-aware**.
5. Swap presets with **Remove Attributes** if you want a different entrance/exit.

---

## Tips and workflow

- **Match text length to preset** — very fast text clips favor snappy slides; longer straps suit slower presets.
- **Pair with Auto Resize Lower Third** if you need graphics that resize to name length ([Auto Resize Lower Third](auto-resize-lower-third/index.md)).
- **Bulk apply** — duplicate a styled text layer to keep the same slide on recurring speakers.
