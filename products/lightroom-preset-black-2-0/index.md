## Install on computer (Lightroom desktop)

1. Extract the `.zip`.
2. Open **Adobe Lightroom**.
3. Click **Presets** (top right) → **⋯** → **Import Presets**.
4. Open the extracted **XMP** folder, **select all** `.xmp` files → **Import**.
5. Presets appear under the **BLACK 2.0** group.
6. Hover a preset and click to apply; use the **amount / intensity** slider to blend strength.

---

## Install on Android

1. Extract the download on your phone (tap the `.zip` → open folder).
2. Open **Lightroom** → **Presets** (bottom left) → **⋯** → **Import Presets**.
3. Navigate to **Downloads** → your extracted folder → **XMP**.
4. Select one file, then **select all** remaining XMP files → **Open**.
5. Open the **BLACK 2.0** group, tap a preset to apply, adjust intensity on the preview slider.

---

## Install on iPhone / iPad

Android-style XMP import is not used here — you import **DNG** files first, then save presets:

1. Extract the zip on your device and open the **DNG** folder.
2. Select **all DNG** files → **Share** → **Lightroom** → **Launch Lightroom**.
3. After import, open the **first** image → **⋯** → **Create Preset**.
4. Create preset group **BLACK 2.0**, name preset **01** (or matching number) → save.
5. Swipe to the next DNG and repeat (**Create Preset** → same group → next number) for every file.
6. Open **Presets** → **BLACK 2.0** to apply; use the intensity control on the preset preview to taste.

## Tips and workflow

- **Intensity slider** — all platforms let you reduce preset strength for softer blacks.
- **Keep XMP on desktop/Android** and **DNG on mobile Apple** — using the wrong folder is the most common install issue.
- **Batch import on desktop** — select all XMP at once instead of importing one-by-one.
